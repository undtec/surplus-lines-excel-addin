Available Functions (1)

  • =SLTAX.SLAPI(calculationType, effectiveDate, stateCode, premiumAmount) Unified function for tax calculations and rate lookups. Use "Tax" or "Rate" for calculationType.

Examples

Calculate Tax (Current Rates):

=SLTAX.SLAPI("Tax", "", "Texas", 10000)

Returns 2 rows: Base Tax, Stamping Fee

Get Current Rate:

=SLTAX.SLAPI("Rate", "", "Florida")

Returns 9 rows: all fee fields

Historical with Fallback:

=SLTAX.SLAPI("Tax", "2020-01-01", "Texas", 10000)

Returns 2-4 rows (includes fallback notice if historical data unavailable)
